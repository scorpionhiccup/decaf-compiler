    Welcome to the Page of Compiler for Decaf Programming Language.

    Lexical Analyser

    Lexical analysis involves the process of analyzing a stream of individual characters, into a sequence of lexical tokens to be fed into the parser. We use Flex for this purpose.

    Parser

    We use bison to analyse that the given input to the compiler, follows the rule as defined by the programming language.
